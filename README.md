# LMS Physics App 🚀

A full Learning Management System for Advanced Level students.

## Features
- 📱 Phone OTP Login
- 👨‍🎓 Student Dashboard
- 👨‍🏫 Teacher Panel
- 📊 Marks Analytics
- 🤖 AI Assistant
- 📝 Quiz System
- 📄 PDF Submissions

## Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- Database: MongoDB

## Setup

### Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
npm run dev
